## 为什么用 MediaLite？

### 网盘本地挂载场景（首要推荐）

115网盘、阿里云盘、OneDrive、百度网盘等支持将文件以本地盘符形式挂载到电脑。挂载后，这些文件在系统中表现为普通本地文件夹，但网盘自带的浏览器普遍难用：缩略图加载慢、不支持视频预览、快捷键缺失。

MediaLite 对挂载网盘的体验与本地硬盘完全一致：

- 缩略图并行生成，首次打开后缓存，再次打开秒加载
- 视频可直接播放，支持字幕、进度记忆、倍速
- 快捷键完整，收藏、筛选、随机浏览一键操作
- 不上传、不联网，文件访问路径完全在本地

**典型用法**：将 115网盘挂载为本地盘符 → 用 MediaLite 打开对应文件夹 → 享受和本地硬盘相同的浏览体验。

### 其他适用场景

- 摄影素材整理（上万张 RAW 预览图快速筛选）
- 视频素材预览（不需要打开播放器逐个确认）
- 本地图集快速浏览与收藏

## 安装与运行

**1. 安装 Python 3.10（64位）**

下载地址：https://www.python.org/downloads/release/python-31011/

安装时勾选 **"Add Python to PATH"**，否则后续命令无法识别。

**2. 安装依赖**

```
pip install PySide6
```

视频缩略图（可选）：安装 ffmpeg 并确保 `ffmpeg` 命令在 PATH 中  
EXIF 信息（可选）：`pip install Pillow`

**3. 启动程序**

下载 `dist_cython/` 文件夹，双击 `启动MediaLite.bat` 运行。  
首次打开文件夹时需要生成缩略图，耗时与文件数量成正比（实测约 14 秒 / 1000 个文件）；缓存生成后再次打开同一文件夹秒加载。

> 源码已编译为二进制（.pyd），不含可读源码。

## 性能说明

- **缩略图缓存**：生成后写入 `~/.cache/medialite/thumbs/`，下次打开同一文件夹秒加载
- **首次加载耗时**：与文件数量成正比，实测约 14 秒 / 1000 个文件（12000 张约 2–4 分钟）；生成期间界面不卡顿，可正常浏览已加载部分
- **并行生成**：最多 4 个线程同时生成缩略图，不阻塞主界面
- **图片解码**：直接读取目标分辨率，不全量解码原图
- **大图加载**：后台线程异步加载，界面不卡顿

---

## 注意事项

- **视频缩略图**：需要 ffmpeg 已安装且在系统 PATH 中，否则缩略图显示灰色占位
- **视频播放**：依赖 PySide6.QtMultimedia，部分格式需系统媒体框架编解码器
- **HEIC / AVIF**：需系统安装对应解码器（Windows 11 已内置 HEIC）
- **GIF**：缩略图在查看器内自动循环播放，不自动旋转
- **缓存清理**：缓存位于 `~/.cache/medialite/`，可手动删除

| 功能 | 状态 |

## 更新记录

### v1.4 · 2026-06-27

**Bug 修复**

- **工具栏随机数量区 ▲▼ 按钮**：修复小字号下 Unicode 三角字符渲染为实心点（`·`）的问题；调整布局顺序为 `[100][张][▲▼]`，数字与单位不再被控件隔开
- **状态栏文字不可见**：修复深色主题下状态栏文案颜色失效（文字与背景同色）
- **查看器属性对话框橙色文字**：修复在大图查看器中打开属性时文字变为橙色的问题；根因为父窗口 stylesheet 级联到子对话框，改用 objectName 作用域限定
- **缩略图界面 Ctrl+I 无效**：Ctrl+I 查看属性快捷键现在在主界面缩略图网格中同样有效（原仅查看器支持）
- **"最近 ▼" 下拉图标显示异常**：修复下拉箭头渲染为实心点的问题（`▾` U+25BE → `▼` U+25BC）
- **全屏按钮文字不显示**：修复 `⛶` 符号在 Windows 字体下不渲染导致按钮文字不可见的问题；同时修复主题切换后全屏按钮文字消失（`_apply_theme` 遗漏该控件）
